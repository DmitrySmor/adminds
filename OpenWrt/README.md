# Скипты list_installed_packages.sh

Выводит список пакетов которые были установлые после установи с сисетмы.
Нужен для поиска пакетов которые установлены пользователем.

Скрипт который выдал ИИ, тоже как вариант

```sh
curl -s https://gist.githubusercontent.com/cvladan/898c3ad14c5ba31806a131ae6bf39e28/raw/list-my-packages.sh | sh
```
